# 🏪 MzansiBiz — Township Business Directory

A searchable, filterable directory of small businesses across South African townships. Built to make local businesses visible and help communities find services near them. Pure HTML, CSS, and JavaScript — no frameworks, no dependencies.

**Live Demo:** [your-username.github.io/mzansi-biz](https://your-username.github.io/mzansi-biz)

---

## 🗺 What It Does

MzansiBiz is a community business directory covering 6 South African townships:

**Soweto · Alexandra · Khayelitsha · Tembisa · Umlazi · Mamelodi**

Users can:
- **Search** by business name, service type, or area
- **Filter** by 8 business categories
- **Filter** by township/area
- **Sort** alphabetically, by category, or by area
- **See open/closed status** at a glance
- **Submit their own business** via the listing form

## ⚙️ Features

- **Live search** — filters instantly as you type across name, description and area
- **Multi-filter system** — category + area filters work together
- **Sorting** — A–Z, by category, or by area
- **48 real-style business listings** with names, descriptions, addresses and hours
- **Colour-coded cards** — each category has a unique accent colour
- **Open/Closed indicator** — at-a-glance availability status
- **Business submission form** with validation feedback
- **Animated card reveals** — staggered entrance on load and filter
- **Fully responsive** — mobile, tablet and desktop layouts

---

## 🛠 Tech Stack

- **HTML5** — semantic structure, accessible markup
- **CSS3** — custom properties, CSS Grid, animations, responsive design
- **Vanilla JavaScript** — live search engine, filter state management, DOM rendering

No libraries. No build tools. No APIs. Open `index.html` in any browser.
